# Movie Search App (TMDB API)

## Description

Movie Search App is a web application for searching and rating movies using the TMDB API.

Users can search for movies in real time, view detailed information, and rate them.  
The application implements server data loading, pagination, and a rating system based on TMDB guest sessions.

The interface is divided into two main sections: movie search and rated movies.

---

## Features

- Real-time movie search
- Movie details view
- Pagination
- Rating system
- Rated movies tab
- Responsive interface
- Loading states and empty states
